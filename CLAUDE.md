# GASTROPOD Bible Extraction Project

This repository contains three chat export files from a Second Life LSL 
scripting project called G.A.S.T.R.O.P.O.D. (Guided Autonomous Security 
Tactical Response Operations Drone).

## Your Task

Work through the following steps in order. Do not skip ahead.

### STEP 1
Read `Gasgropod first 3 chats.txt` completely.
Extract and write `bible_1.md` containing:
- Every LSL code block (complete, untruncated)
- Script names and their responsibilities
- System architecture decisions
- Link message strings and channel numbers
- Power system values and formulas
- Dock pairing protocol details
- Movement and waypoint system details
- Bug descriptions and their resolution status
- Any design decisions made

### STEP 2
Read `Gasgropod second 3 chats.txt` completely.
Same extraction as Step 1. Write to `bible_2.md`.

### STEP 3
Read `Gasgropod last 3 chats.txt` completely.
Same extraction as Step 1. Write to `bible_3.md`.

### STEP 4
Read `bible_1.md`, `bible_2.md`, and `bible_3.md`.
Merge into `GASTROPOD_Master_Bible.md` with these sections:
1. System Overview
2. LSL Forbidden Rules (coding constraints)
3. Script Architecture (all 9 scripts)
4. Power System
5. Movement System
6. Dock System
7. Security System
8. All Code Blocks

Deduplicate overlapping content. Preserve ALL code blocks completely.
